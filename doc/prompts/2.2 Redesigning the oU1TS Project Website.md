### Root Cause
In `index.css`, `.projects-grid-container` was set to `touch-action: pan-y;`. This tells the browser to **only** allow vertical panning (scrolling) via touch gestures and explicitly block horizontal scrolling/swiping (`pan-x`) inside this container.

### Solution
Updated the CSS rule in [index.css](%5BoU1TS%5D/ou1ts.github.io/index.css#L111) to specify `touch-action: pan-x pan-y;` on `.projects-grid-container`. This permits the browser's default touch events for horizontal swipe scrolling to function correctly on mobile screens.
